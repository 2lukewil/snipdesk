Welcome to SnipDesk

A snippet launcher for your browser. Here's the one-minute tour.

Open the launcher anywhere

Press Ctrl+Shift+Space on any page to open SnipDesk. Type to search, then press Enter to drop the snippet into the field you were in.

Go ahead, give it a try.

Nice, that's your shortcut. It opens the launcher on any web page.

Rebind it on Chrome's shortcuts page, and pin the toolbar icon for one-click access.

Organize with folders and tags

Group snippets into folders and tag them. Searching or picking a tag spans every folder. In the launcher, the Left and Right arrows jump between the folder tree and the snippet list.

How fast do you type?

Type the passage below. We'll measure your speed so the saved-time estimate stays honest. Skip if you'd rather not.

Time and money saved

Optional: an hourly wage turns the time you save into a dollar figure in the popup.

Yours, or your team's

Personal snippets work offline with no account. Sign in from the toolbar icon to add the shared team library and cross-device sync. That's optional, and you can do it anytime.

You're set

Create one with New snippet, or right-click selected text on any page and choose Add to SnipDesk. Replay this tour anytime from Settings.
